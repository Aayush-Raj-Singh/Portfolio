# Aayush Raj (); Cyber Security Portfolio

A modern, cyberpunk‑themed portfolio website for **Aayush Raj**, showcasing blue team security focus, automation skills, and hands‑on projects. The site uses an interactive terminal UI, neon hacker styling, and recruiter‑friendly layout.

## Features
- Hacker/terminal aesthetic with neon blue/green palette
- Interactive terminal (bash‑style) with command history
- Responsive layout (desktop sidebar + mobile layout)
- Dedicated pages: Home, About, Projects, Certifications, Contact
- TryHackMe badge embed
- Clean HTML/CSS/Vanilla JS (no frameworks)

## Terminal Commands
- `help`
- `whoami`
- `skills`
- `projects`
- `certifications`
- `contact`
- `clear`
- `date`
- `history`
- `ls`
- `ls skills` (future‑ready resume parser placeholder)

## Quick Start
Open `index.html` in any modern browser.

## Customize
- Update social links in `index.html`, `about.html`, and `contact.html`.
- Update your resume file inside `assets/`.
- Edit terminal output in `script.js`.
- Replace project repository links in `projects.html`.

## Security & Performance Notes
- No API keys stored client‑side.
- Animations are CSS‑based and respect `prefers-reduced-motion`.
- Terminal command handling is local‑only (no data is sent anywhere).
